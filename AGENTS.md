# AGENTS.md

## Contexto

Este é um projeto frontend estático em React + Vite. Não há backend, SDK externo de plataforma ou autenticação.

## Comandos

- Instalar dependências: `npm install`
- Desenvolvimento: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`
- Lint: `npm run lint`

## Publicação

O projeto está preparado para Vercel com saída em `dist/` e domínio previsto `analuizarigueira.com.br`.

## Notas de manutenção

- Preserve o funcionamento estático.
- Não introduza dependências de backend sem decisão explícita.
- Mantenha assets públicos em `public/assets/amagus/`.
- Antes de finalizar mudanças, rode pelo menos `npm run build`.
